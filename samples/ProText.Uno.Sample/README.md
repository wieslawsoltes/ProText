# ProText.Uno.Sample

Minimal Uno/WinUI sample for the `ProText.Uno` implementation.

By default the project references `src/ProText.Uno/ProText.Uno.csproj`, defines `PROTEXT_UNO_AVAILABLE`, and activates the ProText columns through `ProTextBlock`, `ProTextPresenter`, `ProTextBox`, and `ProTextCache`.

Run it with:

```bash
dotnet run --project samples/ProText.Uno.Sample/ProText.Uno.Sample.csproj
```
